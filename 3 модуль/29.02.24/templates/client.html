Car service

All queries should be written within such a pattern:
http://127.0.0.1:4002?...
For example: http://127.0.0.1:4002?id=7

List of query parameters:

Create car by id. Use crt=...

Example: crt=Toyota - creates car with model=Toyota

Get car model by id. Use id=...

Example: id=24 - gets model with id=24

Delete car model by id. Use del=...

Example: del=78 - deletes model with id=78

Update car model by id. Use upd=...

Example: upd=13,Reno - updates model with id=13 to Reno
